# rag-risk（RAG 风险避坑查询）

从知识库中检索目的地常见避坑信息，使用 **RAGRiskAgent** 经结构化 LLM 抽取后，
输出 `RiskOutput` 格式的风险条目数据。

## 触发场景

- 用户有明确的旅行规划意图（包含目的地 + 出行计划）
- 通常与 `rag_experience` 同时调度，优先级相同（并行执行）

## 返回格式

```json
{
  "status": "success",
  "risks": {
    "risks": [
      "西湖周边打车高峰期易堵，若赶班次可能误点，建议提前1小时出发或改乘地铁",
      "龙井茶勿买路边散摊'明前茶'，易以次充好，选大牌茶庄或官方产地门店"
    ]
  }
}
```
